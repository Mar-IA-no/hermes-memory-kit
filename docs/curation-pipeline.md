# Curation Pipeline

Flujo:

1. ingesta y normalización
2. retrieval relacionado
3. inspección opcional del wiki
4. decisión de curación
5. escritura canónica
6. proyección opcional

Salidas posibles:

- solo `evidence`
- `evidence` + links
- `evidence` + nota `library`
- `evidence` + update wiki
- `evidence` + map note
